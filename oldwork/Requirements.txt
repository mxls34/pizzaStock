==========================================
  REQUIREMENTS: Function addStock()
==========================================

FEATURE: ตรวจสอบสินค้าซ้ำ
-----------------------------------------

1. เมื่อเรียก addStock() ให้ตรวจสอบว่าชื่อสินค้า (nameIngredient) มีอยู่ใน Stock array แล้วหรือเปล่า

2. ถ้าชื่อสินค้า**มีอยู่แล้ว**:
   - ให้เพิ่มจำนวน (numberIngredient) เข้าไปในตัวเดิม
   - อัปเดตวันที่เพิ่มเป็นวันปัจจุบัน (formatTime)
   - แสดง alert: "✓ อัปเดต [ชื่อสินค้า] เสร็จสิ้น (จำนวนรวม: [จำนวนใหม่])"
   - ห้ามสร้างรายการใหม่

3. ถ้าชื่อสินค้า**ไม่มี**:
   - สร้างรายการใหม่ (เหมือนปัจจุบัน)
   - ใส่ ID = S{Stock.length}
   - แสดง alert: "✓ เพิ่มข้อมูลเสร็จสิ้น"

4. ทั้งสองกรณี ให้ clear input field ทั้งหมด

==========================================
ตัวอย่าง:
==========================================

Stock = [
  {ID: "S0", name: "เห็ด", number: 5, day: "..."},
  {ID: "S1", name: "หอม", number: 3, day: "..."}
]

ครั้งที่ 1: เพิ่ม "เห็ด" 2 ชิ้น 
  → Update Stock[0].number = 5 + 2 = 7

ครั้งที่ 2: เพิ่ม "พริก" 1 ชิ้น
  → สร้าง S2 ใหม่

=========================================สูตรการคำนวน:
=========================================

1. ค้นหาสินค้า:
   foundIndex = Stock.findIndex(item => item.name === nameIngredient)

2. ตรวจสอบลักษณะ:
   if (foundIndex !== -1) {
      ← มีสินค้าแล้ว
   } else {
      ← ไม่มีสินค้า
   }

3. สูตรเพิ่มจำนวน (ถ้ามีแล้ว):
   Stock[foundIndex].number = 
      parseFloat(Stock[foundIndex].number) + parseFloat(numberIngredient)

4. สูตรสร้าง ID ใหม่ (ถ้าไม่มี):
   idIngredient = "S" + Stock.length

   ตัวอย่าง: 
   - Stock มี 2 รายการ → ID = "S2"
   - Stock มี 5 รายการ → ID = "S5"

5. สูตรวันที่ (เสมอใช้เวลาปัจจุบัน):
   formatTime = new Date()
   
   day = `${formatTime.getDate()}/${formatTime.getMonth()}/${formatTime.getFullYear()} เวลา ${formatTime.getHours()}:${formatTime.getMinutes()}`
   
   ตัวอย่าง: "12/1/2026 เวลา 14:30"
   (หมายเหตุ: getMonth() ฝั่ง 0-11 ถ้าต้องการเดือน +1 ทำ getMonth()+1)
